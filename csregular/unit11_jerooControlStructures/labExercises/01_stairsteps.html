StairSteps

Directions

Write a program that instructs a Jeroo to pick up all the flowers along the stairstep path. To make the program as efficient as possible you must use a while loop in your implementation. The Jeroo should start at location (1,1).

To help you get started I have provided pseudocode for the program.

while not water ahead
   if flower to the right
      turn right
   if flower to the left
      turn left
   pick
   hop
