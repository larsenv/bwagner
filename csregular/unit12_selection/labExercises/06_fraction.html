Fraction

Directions

Write a program that can add, subtract, multiply, and divide fractions. Input should consist of a single line representing a fraction arithmetic problem in the following format:

2/3 + 1/2

The input will consist of fractions with single digit numerators and denominators. Once the problem has been entered the program should perform the arithmetic operation indicated by the operator in the problem.

Step 5: Write Method extractData

The fraction problem is input into the program as a single string value. The extractData method uses the String class's substring method to extract the numerators, denominators, and operator from the problem so the arithmetic operation can be performed.

Assume that a fraction problem is input using the following format:

0
1
2

+
1

The value 2 above is the first numerator in the problem above. To extract this value from the problem, use the following code, which is explained in two parts:

Part 1

num1 = problem.substring(0, 1);     // extract 2 from problem

The code segment above is not complete. The variable num1 is declared as an int. This is necessary so that it can be used in an arithmetic operation, however, this causes a problem since the substring method returns a value of type String. To fix this problem you need to convert the substring into an integer value (ex. convert "2" into 2). To do this you are going to use another Java library class named Integer. The Integer class has a method named parseInt that can do this job for us. The Integer class is similar to the Math class in that its methods are static, therefore you don't need to create an object in order to use them. The code segment above has been modified to include the parseInt method:

Part 2

num1 = Integer.parseInt(problem.substring(0, 1));     // converts "2" into 2

Complete method extractData by extracting num1, denom1, num2, denom2, and operator from the problem. The operator variable will contain a string value.

Step 6: Write Method doArithmetic

You are now ready to do the arithmetic. To determine which arithmetic operation you will need to perform do a comparison using the operator variable. If it contains a "+" you will do addition, if it contains a "-" you will do subtraction, and so forth. Since operator is a String variable you will need to use the equals method to make the comparisons.

Addition and Subtraction

Normally when performing addition and subtraction fraction problems you need to determine the LCM (Least Common Multiply). Unfortunately we have not covered the Java commands that would enable you to perform this task, therefore you are going to use another technique called the Butterfly Method. The following diagram illustrates this technique.

numerator answer = numerator 1 times denominator 2 plus denominator 1 times numerator 2.
denominator answer = denominator 1 times denominator 2.

Multiplication

To multiply fractions you multiply numerator 1 times numerator 2 and denominator 1 times denominator 2.

Division

To divide fractions you cross multiply: numerator 1 times denominator 2 and denominator 1 times numerator 2.
