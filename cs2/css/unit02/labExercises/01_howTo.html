How To Add CSS

As you may remember from our study of HTML, there are three ways to style an HTML document. They include

  • InLine Styling

  • Internal Styling

  • External Styling

We refer to all three of these styling options as CSS or Cascading Style Sheets. When I first learned about CSS I was confused by the word Cascading. What did cascading mean in reference to style sheets? So looked upon the definition of cascading in a dictionary. One definition of cascading is "To occur in a sequence or successive stages".Thefreedictionary.com

From the above definition, it can be inferred that cascading, as it relates to style sheets, means that style sheets are applied in stages one after the other. When an html document utilizes more than one type of styling there is a sequential ordering that determines when the different stylings will be applied to the page.


Below is the order in which the different types of styling will be applied to a HTML document.

  1. Browser Default Styling

  2. External and Internal styling placed inside a HTML page's style tag.
    (They will be applied in the order they are listed in the style tag.)

  3. InLine styling. (Styling inside the HTML tag).


Working through the list from top to bottom. Browser default styling has the lowest priority. Any of these default settings will be overridden by the other forms of styling. External and internal styling have the next priority and they can only be overridden by inline styling. Therefore, inline styling has the highest priority and will override any other styling previously applied to a HTML document.

To demonstrate this "Cascading" of styles, in this lesson you are going to modify the font size style using the different forms of styling and observe their priority ordering.

  4. The starter code above does not include any styling, therefore the browser uses a default style sheet. One of the attributes set in this style sheet is the font size which is equal to 16px. (px = pixels).

  5. View this code in your browser to see the default styling.

    6. Internal Styling

  6. Add the following css code to the HTML document under the comment <!-- Internal Styling -->

    <style>
body {
   font-size: 32px;
}
p {
   font-size: 24px;
}
</style>

    View this code in your browser to see the styling for the font-size change as the internal styling overrides the browser's default settings.

    7. External Styling

  7. Add the following css code to the HTML document under the comment <!-- External Styling -->

    <link rel="stylesheet" type="text/css" href="01_howTo.css">

  8. Create a New File named 01_howTo.css and place it in your CSS folder.

  9. Add the following css code to the CSS file.

    body {
   font-size:8px;
}
p {
   font-size:12px;
}

  10. View this code in your browser to see the styling for the font-size change as the External styling overrides the Internal styling because the link to the css file occurs after the Internal styling, giving the External styling a higher priority.

  11. Cut or remove the External styling section from head section and paste it above the Internal styling section. In other words switch there positions.

  12. View the page in your browser again. The Internal styling should now be applied to the document, overriding the External css, because the Internal styling is defined after the External styling in the head section of the document, giving the Internal style a higher priority.

    13. InLine Styling

  13. Add the following inline css code to the paragraph tag <p>.

    <p style="font-size:48px;">I am using paragraph tag...

  14. View the page in the browser. The font-size of the paragraph should have changed to 48px since inline styling has the highest priority of the three styling techniques and will always override their settings.
