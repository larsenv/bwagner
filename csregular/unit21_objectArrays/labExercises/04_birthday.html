Birthday List

Directions

Write a program that stores birthday information about your friends and family. This information should include the person's name and birthday. The program should allow you to enter info for at least 50 people and then display this information on the console window.

The program should be decomposed into the following two classes: Birthday and BirthdayManager. The Birthday class stores a person's name and birthday. The BirthdayManager class is a client of the Birthday class. It maintains a list of up to 50 people.

In writing these two classes include any necessary instance variables, constructors, and methods.
