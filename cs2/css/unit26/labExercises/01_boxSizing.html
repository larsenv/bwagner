Lab Exercise

  1. Open VS Code and create a new document named 26_1_boxSizing.html and save it to your CSS folder.

  2. Give the document the title: 26_1_boxSizing.

  3. In VS Code create a file named 26_1_boxSizing.css and save it to your CSS folder.

  4. Use external styling to create a document like the one shown below.

  5. Don't forget to add the following html to the head section of your html document:

    <link rel="stylesheet" href="26_1_boxSizing.css">

  6. Use "*" notation to set every element in the document to box-sizing: border-box.

  7. Create three div elements positioned horizontally within the document.

  8. The first div should include the following style properties/attributes:

    • width: 300px

    • height: 100px

    • margin: 20px

    • padding: 5px

    • border: 1px solid black

  9. The second div should include the following style properties/attributes:

    • width: 300px

    • height: 100px

    • margin: 20px

    • padding: 20px

    • border: 5px solid black

  10. The third div should include the following style properties/attributes:

    • width: 300px

    • height: 100px

    • margin: 20px

    • padding: 20px

    • border: 10px solid black
