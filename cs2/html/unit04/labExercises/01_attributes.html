Displaying an Image on a Web Page

There are two sets of instructions for displaying an image on a web page. The first set of instructions are for students who are using an online editor - CodePen. The second set of instructions are for students who are using a desktop editor - Visual Studio Code.

Displaying an Image using an Online Editor (ex. CodePen)

  1. Right-click on the picture of the puppy below and select the option Copy image address.

  2. In CodePen, add an img tag to the body of your html along with a src attribute.

  3. Paste the address of the image you copied in step 1 inside the quotes of the src attribute.


The image should appear in the output window in CodePen.


Displaying an Image using a Desktop Editor (ex. Visual Studio Code)

  1. Right-click on the picture of the puppy and save the image to your HTML folder with the name puppy1.jpg.

  2. Determine the size of the image by following these steps:

    • Open your HTML folder.

    • Right-click on the puppy image icon and select Properties.

    • Select the Details tab.

    • Under the Image section you will find the dimensions of the image.

  3. Use the image tag <img> so that the puppy image is displayed on this page.

  4. Use the src attribute to specify the name and location of image file.

  5. Use the width and height attributes to specify the size of the image.
