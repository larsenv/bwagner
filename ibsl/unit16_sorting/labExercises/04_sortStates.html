NetBeans Workspace: SortStates

JCreator Workspace: SortStates

Directions

Complete the program, SortStates, that sorts a states database into descending order by population.

Implement the State class's compareTo method so that it can compare two populations for ordering. Use the following pseudocode: 

State STATE = (State)OBJ
if getPopulation() = STATE.getPopulation() then
   return 0
end if
if getPopulation() > STATE.getPopulation() then
   return 1
end if
if getPopulation() < STATE.getPopulation() then
   return -1
end if

Below is the array implementation of the bubble sort algorithm.

public void bubbleSort(int[] numbers)
{
	//  Sorts list in ascending order.
	
	int i, temp;
	boolean done;
	do
	{
	   done = true;
	   for(i = 0; i < numbers.length-1; i++)
	   {
		   if (numbers[i] < numbers[i + 1])
		   {            
			   // swap
			   temp = numbers[i];
			   numbers[i] = numbers[i+1];
			   numbers[i+1] = temp;
			   done = false;             // made a swap
		   }            
	   }        
	} 
	while (!done);
}

Modify the bubble sort algorithm so that it can be used to sort the ArrayList of states by population in descending order. You will need to use the following ArrayList methods in your implementation:

  • Object get(int index)
  • void set(int index, Object element)

You will also need to use the State class's compareTo method

  • int compareTo(Object obj)

The temp variable in the bubble sort algorithm needs to be changed to type State.
