14-2 Address Class

Person class

Below is a modified version of the Person class discussed in lesson 1.

class Person
{
    // constructor initializes object variables
    constructor(firstName, lastName, address)
    {
        this.firstName = firstName;
        this.lastName = lastName;
        this.address = address;
    }
    
    // method getFullName returns the person's name firstName lastName format
    getFullName()
    {
        return this.firstName + " " + this.lastName;
    }
    
    // method getFormalName returns the person's name in lastName, firstName format
    getFormalName()
    {
        return this.lastName + ", " + this.firstName;
    }
    
    getMailingAddress()
    {
        return this.getFullName() + "<br>" +  this.address.getAddress();
    }
}

The class includes the following modifications:

  1. The constructor includes another field named address and the parameter list includes a parameter variable named address as well.

  2. A new method has been added to the class named getMailingAddress. This method makes a call to a method named getAddress that is a member of a class named Address you will be developing later in the lab exercise.
