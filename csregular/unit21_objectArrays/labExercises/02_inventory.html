BookInventory

Directions

An English class in a small rural school is reading Hamlet. The teacher assigns a book to each student in the class. She has asked you to write a program that can help her keep track of books assigned to each student. You decide the problem can be decomposed into the following two classes: Book and Inventory. The Book class stores the name of a student and the assigned book number. The Inventory class is a client of the Book class. It uses an array to store up to 20 books.
