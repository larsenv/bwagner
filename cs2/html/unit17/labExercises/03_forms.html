Lab Exercise

  1. Open VS Code and create a new document named 17_3_forms.html and save it to your HTML folder.

  2. Give the document the title: 17_3_forms.

  3. Use form elements and attributes to create the document shown below.

  4. Use the password input type for the password text box.

  5. Use the span element for the username and password labels then use the following internal styling to format them:

    <style> span.text { font-family: consolas; } </style>

  6. Don't forget to assign the class attribute for both spans to text.

  7. Use inline styling to set the width of the fieldset element to 300px.

  8. Use the button element for the Ok and Cancel buttons.
