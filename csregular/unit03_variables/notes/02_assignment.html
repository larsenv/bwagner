Assignment Operator (=)

In Java, the equal sign (=) is called the assignment operator. The assignment operator is used to assign a value to a variable. Here are some examples.

// declare variables

int integerValue;
double floatValue;
char charValue;


// assign values to variables

integerValue = 875;
floatValue = 28.975;
charValue = 'b';

The assignment occurs from right to left. The value on the right is copied into the variable on the left.

The assign operator can be used to replace the contents of a variable with a new value.

// declare variable

int num;


// assign values to variable

num = 0;
System.out.println(num);
num = 50;
System.out.println(num);
num = 225;
System.out.println(num);

Here is the output. Notice how the variable's value changes each time the assignment operator is used.

0
50
225
Programming Note: You can include comments in your code by placing them after two forward slash symbols //. Here is an example from the code above.

// declare variable

Comments are ignored by the compiler and do not affect program execution.

Initialize Variable

You can declare and assign a value to a variable at the same time. This is called initializing a variable.

int num = 500;
double price = 89.25;

It is good practice to always initialize your variables when you declare them. If you don't know what the value for a variable should be when you declare it assign it the data type's default value. The default value for int and double is zero.

int num = 0;
double price = 0;

You can declare multiple variables of the same type, on one line, by separating them with commas.

int game1, game2, game3;

You can initialize multiple variables of the same type, on one line, by separating them with commas.

int game1 = 5, game2 = 7, game3 = 12;

Variable Output

To display the contents of a variable you simply include the variable name in a println statement without double quotes around it. If you place double quotes around the variable it will display the variable's name not its contents.

int average = 95;

System.out.println("average");
System.out.println(average);
average
95

Labeling Output

The term user-friendly is often associated with computer software. A user-friendly program is one that is easy for a user to use and understand. One aspect of a user-friendly program is the way in which the output of the program is displayed on the screen. Look at the sample program below and the output produced.

int temperature = 98;

System.out.println(temperature);
System.out.println();
System.out.println("The high temperature today is " + temperature);
98

The temperature today is 98

Which of the two output lines is more user-friendly. Well, obviously the second one, because it describes or labels the output so the user knows the values context. You should always strive to make your programs as user-friendly as possible.

Concatenation Operator

In Java, the plus sign (+) is used to label the data within a println statement. When the plus sign is used in this context it is referred to as the concatenation operator. Let's look at some examples.

int testGrade = 87;
int weight = 195;
double cost = 29.95;

System.out.println("Test grade = " + testGrade);
System.out.println("You can bench press " + weight + " pounds.");
System.out.println("The cost of the shirt is " + cost);
Test grade = 87
You can bench press 195 pounds.
The cost of the shirt is 29.95

The concatenation operator can also be used to provide spacing between data items.

int num1 = 10;
int num2 = 20;
int num3 = 30;

System.out.println(num1 + " " + num2 + " " + num3);
System.out.println(num1 + "     " + num2 + "     " + num3);

10 20 30
10     20     30
