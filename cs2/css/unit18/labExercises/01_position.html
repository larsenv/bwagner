Position

In this exercise, you learn how to use the position property to control how elements are placed within the browser window. The position property provides 5 different position values:

  • static

  • relative

  • absolute

  • fixed

  • sticky

By default, the browser will position all elements on the screen using the static option.

If you choose to use one of the other options then you will also need to use one or more of the following properties: left, right, top, and bottom to specify exactly where the elements should be placed within the window.

In this exercise, we will only be using two of the options: absolute and sticky.

    6. Absolute Positioning

    To demonstrate absolute positioning you are going to add a logo to the header section of the page.

  6. Right-click on the following image and select Save image as.. and save the image to your CSS folder.

  7. Add the following html to body of the web page inside the header tag just below the title "Denton High School".

    <img id="logo" src="bronco.png" style="width: 8%">

  8. View the web page in the browser. Notice how the logo is position right after the title. Let's use absolute positioning to move the logo to the far right side of the header and center it vertical within the header.

  9. Add the following internal styling to the style section of your page.

    #logo {
   position: absolute;
   right: 15px;
   top: 6px;
   height: 100px;
}

  10. View the page in the browser. Absolute positioning allows you to place an element inside its container, in this case, the header, by specifying the element's location relative to its container. For example, top:0 and left:0 would be located at the top right corner of the header element, not the window. By choosing top:8px and right:15px the logo is positioned 8 pixels from the top of the header and 15 pixels from the header's right edge.(Typically when deciding on a location, you would choose either top or bottom and then left or right.)

    Sticky Positioning

  11. View the web page in the browser. Scroll the window vertically. Notice how the nav bar scrolls off the top of the screen.

  12. Add the following internal styling to the style section.

    #menu {
   position: sticky;
   top: 0;
}

  13. View the web page in the browser. Scroll the window vertically. Notice now how the nav bar stops scrolling when it reaches the top of the screen. Sticky positioning can best be described as a combination of relative positioning and fixed positioning. The element, in this case, the nav element, is initially positioned on the screen at a specific (relative) location. Its location can be changed by scrolling until it reaches the value specified in the top property in which case it becomes fixed at that location until the scrolling is reversed.
