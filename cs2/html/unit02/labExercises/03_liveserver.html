Lesson

Lesson 2-1 stated that there were two ways to display a web page locally in a browser without first having to upload it to hosting site on the Internet. In this lesson you are going to use the second method of a displaying an HTML file, which uses a local web server, also know as a development server.

Live Server

Live Server is a simple web server that you can install as an extension in VS Code. (Note: Live Server was one of the extensions you were asked to install in the previous lesson.)

Follow these steps to view an HTML file using Live Server.

  1. First the HTML file to be displayed must be open an editor window. If not already open, open the file MyFirstHTML.html created in lesson 2-1 so that it appears in an editor window.

  2. Click the Go Live button located in the editor's status bar.

    Live Server will launch a browser and send it the web page to be displayed. Any changes made to the HTML file while the server is running will automatically be updated in the browser window.

    You can also start Live Server by right-clicking on the HTML file you would like to display and select Open with Live Server.

  3. To stop Live Server click the button in status bar again which is now displaying the port number which the server is using to connect to the computer. If you would like to know about ports click the following link: Port Numbers Used for Computer Networks
