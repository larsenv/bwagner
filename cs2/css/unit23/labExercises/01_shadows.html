Lab Exercise

  1. Open VS Code and create a new document named 23_1_shadow.html and save it to your CSS folder.

  2. Give the document the title: 23_1_shadow.

  3. In VS Code create a file named 23_1_shadow.css and save it to your CSS folder.

  4. Use external styling to create a document like the one shown below.

  5. Don't forget to add the following html to the head section of your html document:

    <link rel="stylesheet" href="23_1_shadow.css">

  6. View the link on rounded corners then create two content boxes that are similar to the ones shown in the document below. Use the float attribute to place the boxes side by side.

  7. Use a text-shadow effect for the heading title.

  8. Use a box-shadow effect for two content boxes.

  9. Use an increasingly transparent radial gradient for the content background. The orange color is number: 255 140 0 (decimal) or FF C8 00 (hex).
