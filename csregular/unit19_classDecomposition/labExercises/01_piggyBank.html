Piggy Bank

Directions

In this assignment we are going to write a program that simulates the operation of a piggy bank. A piggy bank allows you to enter coins through a slot in the top. When the piggy bank is full you can retrieve the money by smashing it on the ground or hitting it with a hammer.

Using class decomposition we are going to decompose this problem into two classes. The first class will be named PiggyBank. Its job will be to store the money that is inserted into the bank. The second class will be named PiggyBankManager. Its job will be to manage the program by creating a PiggyBank object that it can send messages to asking it to store the coins as they are inserted into the bank. This class will contain the main method for the program.

Implement PiggyBank Methods

  1. Implement the methods insertQuarter, insertDime, insertNickel, and insertPenny so that each method's corresponding instance variable is incremented by one.

    For example, the job of the insertQuarter method is to allow a user to add a quarter to the bank. To provide this capability the insertQuarter method should include code that increases the quarters variable by one.


  2. Implement the method smashBank. The purpose of this method is to allow a user to see how much money has been added to the piggy bank. To do this the smashbank method needs to add the total number of cents that the bank has collected. To calculate the total cents use the following formula:
