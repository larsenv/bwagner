Description

There are four ways to select and use a font on a web page.

  1. Use a font that is already installed on your computer by the operating system.

  2. Download a font from the Internet and install its files on your computer.

  3. Download a web font and embed the font on your server. (In our case, the folder where your html files reside.)

  4. Use an online web font service like Google Fonts. With the Google font tool, you can select a font embedded on Googleâ€™s servers, and you can copy the code that makes the font available without downloading. All fonts on the Google Fonts website are free to use.


    5. Since installing fonts on school computers is blocked by admin, you will only be learning options 3 and 4 in this exercise.

    Web Fonts

  4. Read the article from Mozilla entitled Web Fonts.

  5. The article has a section entitled "Active learning: A web font example". Follow the steps to learn how to embed a font and how to use a font from an online service.

    Select and download two fonts from Fonts Squirrel. Use one of the fonts for the heading "Embedded Fonts" and other for the first paragraph. The heading font should be a san serif font and the paragraph font a serif.

    6. Online Fonts

  6. Select and copy the code for two fonts from Google Fonts.

    Again select one font for the heading and another for the paragraph.

    Adding an online font to your web page requires two steps. Step 1: Add a link tag to the head section of your html document with Google page that contains the font information. Here is an example:

    <link href="https://fonts.googleapis.com/css?family=Roboto&display=swap" rel="stylesheet">
    Step 2: Add the font-family property to your web pages styling that references the name of the Google font you are using. Here is an example: 
    font-family: 'Roboto', serif;
