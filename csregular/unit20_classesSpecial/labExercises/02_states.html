States

In this assignment you are going to implement two classes: a State class and a UnitedStates class. The UnitedStates class will be a client of the State class. It will also contain the main method for the program.

Directions

State class

The State class will be used to store information about a state including its state bird, flower, motto, song, and tree.

Create a file named State.java and add the following code.

Instance Variables

We will need a variable to store the state's name, bird, flower, song, motto, and tree. Add the following code to the instance variables section.

private String name;      // The state's name
private String bird;      // The state's bird
private String flower;    // The state's flower
private String song;      // The state's song
private String motto;     // The state's motto
private String tree;      // The state's tree

Constructors

  1. Create a default constructor for the State class. The default value for a string is "" (null string).
  2. Create a second constructor for the State class that will allow a client to assign specific values to each of the 6 instance variables.

toString Method

A toString method provides clients an easy way to print the contents of the instance variables of objects defined within the client program. Here is our toString method defined for the State class. Add this code to the class under the toString section. 

public String toString()
{
   return "State Name   : " + name + "\n" +
          "State Bird   : " + bird + "\n" +
          "State Flower : " + flower + "\n" +
          "State Song   : " + song + "\n" +
          "State Motto  : " + motto + "\n" +
          "State Tree   : " + tree;
}
