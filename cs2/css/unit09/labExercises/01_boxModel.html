Lab Exercise

  1. Open VS Code and create a new document named 9_1_boxModel.html and save it to your CSS folder.

  2. Give the document the title: 9_1_boxModel.

  3. Read both the BoxModel and Outline articles at w3schools.com then continue with the instructions below.

  4. Create a webpage similar to the sample screen below. Create a h1 element in the body of the webpage with the following text: "Happy Holidays". Add styling to the body tag and the h1 tag. Here are the property/attributes used to create the sample below.

    • body background image: "pattern.gif", repeated
    • h1 outline: size=20px, type=solid, color=green
    • h1 border: size=3px, type=solid, color=black
    • h1 margin: auto (will center h1 element)
    • h1 padding: 20px
    • h1 width: 500px
    • h1 height: 100px
    • h1 font size: 4.0em
    • h1 text color: white
    • h1 background color: red

You will need to use a property/attribute we have not covered yet called text-align. This property is used to center text horizontally within its container, in this case the h1 element.

text-align: center
