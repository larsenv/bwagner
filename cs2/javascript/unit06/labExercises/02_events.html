        <div id="dialog"><h3>Do your feet stink?</h3>
            
       </div>
        <div id="response"><h1>Just Kidding!</h1></div>
        <script>
            function moveDialog()
            {
                var w = window.innerWidth;    // width of body
                var h = window.innerHeight;   // height of body
                var left = Math.floor(Math.random() * (w - 400));
                var top = Math.floor(Math.random() * (h - 150));
                document.getElementById("dialog").style.left = left+"px";
                document.getElementById("dialog").style.top = top+"px";
            }
            
            function closeDialog()
            {
                var elem = document.getElementById("dialog");
                elem.remove();
                document.getElementById("response").style.display = "block";
            }
        </script>
    </body>

  3. The div with id dialog is styled to be a dialog box centered in the middle of the document. The function moveDialog repositions the dialog box at at random location within the boundaries of the document. The function closeDialog removes the dialog box from the document making it disappear.

    Add two buttons to the dialog box. The first button should be labeled "Yes" and the second button should be labeled "No". The "Yes" button should call the function closeDialog when it is clicked and the "No" button should call the function moveDialog when the mouse moves over the top of it. The styling for the buttons has been provided.
