Pythagorean Theorem

Directions

The Pythagorean Theorem states that, in a right-angled triangle, the square of the length of the hypotenuse is equal to the sum of the squares of the lengths of the other two sides.


Write a program that allows a user to enter in two sides of a right triangle. Then use the Pythagorean Theorem to calculate and display the hypotenuse of the triangle. The hypotenuse is the longest side of the triangle. Use the Math class's sqrt and pow methods in your implementation.

Round the answer to the nearest integer. Use the Math class's round method. Depending upon your implementation you may need to cast the result returned by the round method to an int.

Implementation

This program can be implemented in four steps.

Step 1: Declare variables

Declare three variables named a, b, and c. Variables a and b should be declared type int and variable c should be declared type double. Initialize all three variables with a value of zero.

Step 2: Read input from keyboard

  • Add Scanner class import statement at the very top of the program.

    import java.util.Scanner;

    or use the shortcut version

    import java.util.*;

  • Add a line of code to the variables section that instantiates a Scanner object.

    Scanner keyboard = new Scanner(System.in);

  • Prompt user for the length of two sides of a triangle: a and b. Use Scanner class's nextInt method to read input values.

    • System.out.print("Enter side 1 -->");   // prompt
a = keyboard.nextInt();                 // read a's length
System.out.print("Enter side 2 -->");   // prompt
b = keyboard.nextInt();                 // read b's length

Step 3: Use Pythagorean Theorem to calculate the length of the hypotenuse c

Use the Math class's sqrt and pow methods to construct the formula.

c = Math.sqrt(Math.pow(a, 2) + Math.pow(b, 2));

Step 4: Display the result of the calculation

Round the value of hypotenuse c to the nearest integer using the Math class's round method.

System.out.println();
System.out.println("The hypotenuse of the triangle = " + Math.round(c));
