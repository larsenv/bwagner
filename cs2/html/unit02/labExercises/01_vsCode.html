Displaying a Web Page in a Browser Window

Normally the goal of writing a webpage is to display it on the Internet, however when first designing and coding a website your testing should be done locally on your computer. There are two ways that a webpage can be displayed locally:

  1. Navigate to the folder on your local computer where the webpage is saved and double-click on the HTML file to launch the webpage in a browser window.

  2. Install a web server on your local computer then launch the web server and have it send the webpage to the browser where it is then displayed.

    3. A web server is program or computer that stores websites on the Internet and delivers web pages to viewers upon request. This service is referred to as web hosting. Every web server has a unique address, called an Internet Protocol address, that tells other computers connected to the Internet where to find the server on the vast network.

    When you install a web server program locally on your computer, you can assign it the IP address 127.0.0.1, also known as localhost. This IP address will allow you to use the webserver without connecting it to the Internet. All data requested from this server will be sent to your local browser. If you would like more information on this topic click on the following link: WhatIsMyIPAddress.com.
