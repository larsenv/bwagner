Resources

  • HTML Plug-ins

  • HTML YouTube

Lab Exercise

  1. Open VS Code and create a new document named 18_3_youtube.html and save it to your HTML folder.

  2. Give the document the title: 18_3_youtube.

  3. Use the iframe element to play one of your favorite youtube videos within your document.

  4. To get the width and height of your video right-click on your youtube video while viewing it at YouTube and select Stats for nerds.

  5. The video url must be in this format: https://www.youtube.com/embed/video id

  6. If you don't have a favorite you can use this one: https://www.youtube.com/embed/nKIu9yen5nc"

  7. After viewing the video in your document modify your code so that it uses the autoplay feature.
