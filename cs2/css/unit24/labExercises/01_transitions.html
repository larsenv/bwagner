Lab Exercise

  1. Open VS Code and create a new document named 24_1_transitions.html and save it to your CSS folder.

  2. Give the document the title: 24_1_transitions.

  3. In VS Code create a file named 24_1_transitions.css and save it to your CSS folder.

  4. Use external styling to create a document like the one shown below.

  5. Don't forget to add the following html to the head section of your html document:

    <link rel="stylesheet" href="24_1_transistions.css">

  6. Copy the following image to your HTML images folder:

  7. Add the kitten image three times to your document positioned horizontally with a margin of 15px between each image.

  8. Assign the first image the id kitten1, the second kitten2, and the third kitten3.

  9. Add the following css so that the first image's styling can be modified when the mouse hovers over the image: 

    #kitten1:hover  
{

}

    Repeat for kitten2 and kitten3.

  10. Add styling to kitten1 so that when the mouse hovers over it the following property/attributes are activated:

    • transform: translate(0, 300px);

    • transition: transform 3s;

    • transition-timing-function: ease;

  11. Add styling to kitten2 so that when the mouse hovers over it the following property/attributes are activated:

    • transform: rotate(360deg);

    • transition: transform 4s;

    • transition-timing-function: ease-in;

  12. Add styling to kitten3 so that when the mouse hovers over it the following property/attributes are activated:

    • transform: rotateY(360deg);

    • transition: transform 5s;

    • transition-timing-function: ease-out;
