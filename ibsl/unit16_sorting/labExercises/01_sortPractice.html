NetBeans Workspace: SortPractice

Directions

The starter code in this exercise includes both the selection sort and bubble sort algorithms. The selection sort is implemented so that it sorts an array of Strings and the bubble sort is implemented so that it sorts an array of integers.

Add code to each of the sorting algorithms so that each pass through the array is displayed on the console. A pass through the array is defined as one iteration through the inner loop.

For example, suppose you have an array consisting of 5 random integers, one pass through an array using the selection sort algorithm would result in the following changes to ordering of the array:

Original Array: 9 4 7 2 6
1st Pass      : 2 4 7 9 6

The selection sort starts by traversing the entire array from beginning to end searching for the smallest value. When it reaches the end of the array it swaps the current number 9, located at index 0, with the current smallest number 2. The number 2 is now in its proper position in the array after 1st Pass.

A 2nd pass through the array would result in the following changes to the array:

Original Array: 9 4 7 2 6
2nd Pass      : 2 4 7 9 6

There were actually not any changes to the array with the 2nd pass because the number 4 was the next smallest number in the array, so after traversing the list starting at index 1 the value 4 was replaced with the value 4. After 2nd pass the numbers 2 and 4 are now in their correct positions in the array.

A 3rd pass through the array produces the following changes to the array:

Original Array: 9 4 7 2 6
3rd Pass      : 2 4 6 9 7

During the 3rd pass through the array the algorithm started traversing the array starting at index 2. With two more passes through the array it will be sorted.
