Lesson

In this lesson your will learn an important concept in html structure called display type. Most html tags fall into one of two display types or categories. These tags by default are either block type or inline type. Each display type has its own display properties.

Block Type

A block type element always starts on a new line and extends to take up the full width of the window.

Here are some examples of block elements you have already used: p, pre, and h1-h6.

In this lesson, a new block tag is introduced called the div element. The div element is important because it is used as a container for other html elements.

Inline Type

A inline type element does not start on a new line and will only take as much width as necessary.

Here are some examples of inline elements you have already used: a, br, strong, em, and img.

In this lesson, a new inline tag is introduced call the span element. The span element is used as a container for text. Combined with CSS it is often used to style parts of text.
