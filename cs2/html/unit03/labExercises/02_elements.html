Lab Exercise

Notice: The instructions outlined in steps 1-3 below where written for students using the Visual Studio Code editor in my computer lab at school. If, however, you are doing these assignments online. You will need to use the instructions outline in the previous lesson for creating a new pen in CodePen. Since CodePen automatically saves your work to its cloud storage you do not have the ability to save files with a name. For convenience, I have included those instructions below.

  1. Open CodePen, if not already open. You may have to Sign In, but more than likely it will have remembered that you were logged in after you setup your account.

  2. After CodePen loads, click on the create new Pen button on the dashboard.

  3. Once the new pen appears, click on the ChangeView button to change the editor layout to the one shown below highlighted in white.

  4. Copy the the HTML into CodePen's HTML editor.

  1. Open your CS2 folder in VS Code

  2. Create a new document named 3_2_elements.html and save it to your HTML folder.

  3. Copy the following starter code into your html document.

    <!DOCTYPE html>
<html>
<head>
   <title>3_2_elements</title>
</head>
<body>
   
</body>
</html>

  4. Add the following code to the body of your html document.

    • The heading (use h1 tag) What I did for Summer Vacation.

    • A paragraph (use p tag) describing what you did over the summer break. You can't say nothing.

    • There needs to be at least two paragraphs.

  5. Add the following code right above the body element: 

    <head>
   <title>3_2_elements</title>
</head>
