Lab Exercise

    Notice: The instructions outlined in steps 1-3 below where written for students using the Visual Studio Code editor in my computer lab at school. If, however, you are doing these assignments online. You will need to use the instructions below.

    1. Open CodePen, if not already open. You may have to Sign In, but more than likely it will have remembered that you were logged in after you setup your account.

    2. After CodePen loads, click on the create new Pen button on the dashboard.

    3. Copy the HTML below into CodePen's HTML editor.

  1. Open your CS2 folder in VS Code.

  2. Create a new document named 5_1_paragraphs.html and save it to your HTML folder.

  3. Copy the following starter code into your html document.

    <!DOCTYPE html>
<html>
<head>
    <title>5_1_paragraphs</title>
</head>
<body>
      
</body>
</html>

  4. Click on the View menu and select Toggle Word Wrap.

  5. Use the h1 tag to format Main Heading.

  6. Use the h2 tag to format Section 1 and Section 2 sub-headings.

  7. Use the p tag for the paragraph in section 1. For the text of the paragraph use loremipsum dummy text.

  8. Use the p at beginning and end of the paragraph in section 2.Use br tags at the end of each sentence (after a period) so that each sentence starts on a new line. For the text of the paragraph use loremipsum dummy text.

  9. Click on the following link: Famous Poets and Poems

  10. Copy the title and poem for The Road Not Taken by Robert Frost.

  11. Use the h3 tag for title and poet name.

  12. Use the pre tag for the poem so that the poem is displayed in the same format as the Famous Poets and Poems site.
