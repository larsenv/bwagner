light

flight
flag
fit
high

fast
shaft

System.out.println(str.substring(0, 3) + str.substring(9));

Part 2

Using the substring "light", that you extracted from "flashlight", form the following words:

lights
lighter
lightning
enlightened

Use the concatenation operator (+) to combine the substring "light" with several string literals to form the new words. Recall that a string literal is a string that you form by enclosing a series of characters inside double quotes. (Example: "ening" is a string literal)
