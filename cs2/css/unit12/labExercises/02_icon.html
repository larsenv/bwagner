    Lab Exercise

    1. Open VS Code and create a new document named 12_2_icon.html and save it to your CSS folder.

    2. Give the document the title: 12_2_icon.

    3. Add the following starter code to the html document.

      <!DOCTYPE html>
<html>
<head>
      <title>Icon2</title>
      <script src="https://kit.fontawesome.com/ce5f59b212.js" crossorigin="anonymous"></script>
      <link href="https://fonts.googleapis.com/icon?family=Material+Icons"
      rel="stylesheet">
      <style>
         body {
            margin: 20px; 
         }
         #b1 {
            font-size: 4.0em;
            color:red;
            border: 5px solid black;
            border-radius: 20px;
            padding: 10px;
         }
         #b1:hover {
            cursor: pointer;
            background-color: blue;
         }
      </style>
</head>
<body>
   <button id="b1" onclick="window.open('https://www.texashealth.org/denton/','_blank');">
      <i class="fas fa-ambulance"></i></button>
   
</body>
</html>

    4. Display the html document in a browser. Hover the mouse over the ambulance icon and then click the mouse button. As you can see, one of the fun uses of icons is to create your own custom buttons.

    5. Using the html and css for the ambulance button create your own button next to the ambulance button with an icon selected from one of the icon libraries.

    6. Your button should be similar to the ambulance button, but also should have your own custom styling, such as different colors, border-size, padding, and margins.

    7. Create a button tag in the body section of the html with the id "b2".

    8. Link the button to a website so that when a user clicks on the button the site will appear in a new tab.
