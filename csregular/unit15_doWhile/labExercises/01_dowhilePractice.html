Directions

Write a program that includes 4 while loops that meet the following criteria:

  • Use a do while loop to print the numbers from 0 to 20 across the screen with a space between each one.
  • Use a do while loop to print the odd numbers from 1 to 50 across the screen with a space between each one.
  • Use a do while loop to print the numbers from 0 to 200 in increments of 10 across the screen with a space between each one.
  • Use a do while loop to print the lowercase alphabet from the ASCII chart. Recall that all the letters and symbols on the keyboard are assigned numbers (digitized).For reference, here is the ASCII chart.




    To convert a number to its character representation cast the integer to a char. Here is an example: 
    System.out.println((char)97);   // prints the letter 'a'
