Width vs Max-Width

In this exercise, you are going to learn the difference between width and max-width.

Lab Exercise

  1. Open VS Code and create a new document named 17_1_maxWidth.html and save it to your CSS folder.

  2. Right-click on the two images below and select Save Image As... Save the images to your CSS folder.

       

  3. Add the following starter code to your html document:


    <!DOCTYPE html>
<html>
   <head>
       <title>max-width</title>
       <style>
          .container {
             height: 350px;
             display:flex;
             flex-direction: column;
             align-content: space-between;
          }
          #puppy1 {
            width: 400px;
            border: 4px solid black;
          }
          #puppy2 {
            width: 400px;
            border: 4px solid black;
          } 
       </style>
   </head>
   <body>
      <div class="container">
         <img id="puppy1" src="Bloodhound.jpg"></div>
         <img id="puppy2" src="Redbone.jpg"></div>
      </div>
   </body>
</html>

  4. View the page in the browser. You should see two images of hound dog puppies. Both of these images have been assigned a width of 400px. When you use the width property to assign a size to an element, in this case an image, the size is fixed and the elements size cannot be changed regardless of what is going on around it. To demonstrate this feature, resize the browser window so that the two images are covered over by the right border of the browser's window. Notice that neither of the images change their size. Instead the browser adds a horizontal scroll bar at the bottom of the window so that the images can be viewed in their entirety.

  5. In the internal styling section locate puppy1. Change the value of the width property from 400px to 100%. By using a percentage for the size the image's width the image size is no longer fixed and it will now stretch to cover 100% of the browser window. If the window is resized, the image will also resize so that it maintains the 100% coverage of the window. To demonstrate this feature, again resize the browser window. Notice that the image shrinks and stretches to maintain full coverage of the window.

  6. Change the width size of puppy1 to 50% and resize the browser window again. Notice that the puppy image maintains its size relationship with the window.

  7. Change the width property of puppy1 back to 400px.

    8. From the demonstrations above, you learned that the width property can have a fixed size or a size that is a percentage of its containers width. Both of theses scenarios are problematic for web developers who wish to use the Responsive Web Design model. If a web page containing a fixed width image is displayed on a mobile device, such as a cell phone, the image might be larger than the screen size of the device which would force the user to scroll horizontally to see the image. On the other hand, if the image width is assigned a percentage value then it becomes difficult to manage its size. For example, you may not wish for an image to always cover the entire browser window.

    To fix these problems, CSS has a property named max-width. Lets add some CSS code to the web page to see how max-width can help.

  8. Locate the styling for puppy2 and then change the width property from 400px to 100%.

  9. Add the following line of code to puppy2's styling: max-width: 400px;

  10. Open the web page in the browser, then resize the browser window so that it's right edge covers over the puppy images. Notice how the puppy2's width does not change until the browser window's width is smaller than 400px.

First we set the puppy2's width to 100%. This normally would make the image stretch to cover the entire browser window. However, by setting the max-width property to 400px we set a maximum size for the image to 400px. The max-width property allows for the image to be smaller than 400px but it can never be larger than 400px. The image did not start scaling its size until the browser window was smaller than the max-width value of 400px.
