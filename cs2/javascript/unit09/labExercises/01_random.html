  3. Display the html file in a browser and click on the drop-down list.

  4. The drop down list displays a series of numbers from 10 to a million. When the user presses the generate number button the web page displays a random number between 1 and the number selected from the drop-down list. For example, if the user selects 100 from the drop-down list then the page would display a random number between 1 and 100 inclusive.

  5. Complete the function doRandom. This function retrieves the selected number from the drop-down list, generates a random number from 1 up to that number, then displays it in the "output" div.

    To complete the function follow these steps:

    • Use document.getElementById to get a reference to the drop-down list with id "ranges" and store the reference in a variable named range.

    • Retrieve the number selected by the user in the drop-down list and store it in a variable named selection. One way to get the selected number is to use the drop-down list's value attribute.
      Example: range.value

    • Use the math function parseInt to convert the value in the variable selection from a string to an integer and store the value in a variable named num.

    • Use the math function random to create a random number in the range of 1 to num and store the value in a variable named random.

    • Display the random number in the "output" div. Use the math function round to display the random number rounded to an integer.
