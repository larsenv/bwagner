Lab Exercise

  1. Create a new folder in your CS2 folder named Javascript.

  2. Open VS Code and select Open Folder... from the File menu.

  3. Navigate to your Javascript folder and click the button Select Folder.

  4. Create a new HTML file named 1_1_output.html and save it to your Javascript folder.

  5. Copy the following code to your source file:

    <html>
    <head>
        <title>1_1_output</title>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    </head>
    <body>
        <div id="demo"></div>
        <script>

        </script>
    </body>
</html>

  6. Add a line of code inside the script tag that will display the following text using an alert window:
    "This is displayed using an alert window"

  7. Add a line of code inside the script tag that will display the following text using the document.write command:
    "This is displayed using <b>document.write</b>"

  8. Add a line of code inside the script tag that will display the following text using the document.getElementById().innerHTML command:
    "This is displayed using <b>document.getElementById.innerHTML<b><br><br>"
