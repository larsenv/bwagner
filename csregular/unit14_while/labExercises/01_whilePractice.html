While Practice

Directions

Write a program that includes 5 while loops that meet the following criteria:

  • Use a while loop to print the numbers from 1 to 25 across the screen with a space between each one.
  • Use a while loop to print the numbers from 50 to 70 across the screen with a space between each one.
  • Use a while loop to print the numbers from 0 to 100 in increments of 5 across the screen with a space between each one(hint: use += ).
  • Use a while loop to print the numbers from 20 down to 1 across the screen with a space between each one.
  • Use a while loop to print the string "computer" 15 times down the screen.
