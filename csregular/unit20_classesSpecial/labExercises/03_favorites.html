Favorites

In this assignment you are going to implement two classes: a Favorites class and a FavoritesManager class. The FavoritesManager class will be a client of the Favorites class. It will also contain the main method for the program.

Instance Variables

Add the following line of code under the instance variables section.

Since the Favorites object used in this class is accessed both in the inputFavorites method and the printFavorites method it has to be declared as an instance variable. Notice that the Favorites object is only declared. An object has not been instantiated and assigned to it yet. This will take place in the inputFavorites method after the information for the object has been input from the keyboard.

inputFavorites Method

The purpose of the inputFavorites method is to allow a user to enter his/her favorite things from the keyboard and then store this information in a Favorites object. Add the following code to the inputFavorites method.

Scanner keyboard = new Scanner(System.in);

System.out.print("Enter person's name -->");
String name = keyboard.nextLine();
System.out.print("Enter favorite food -->");
String food = keyboard.nextLine();
System.out.print("Enter favorite color -->");
String color = keyboard.nextLine();
System.out.print("Enter favorite music -->");
String music = keyboard.nextLine();
System.out.print("Enter favorite automobile -->");
String auto = keyboard.nextLine();
System.out.print("Enter favorite sport -->");
String sport = keyboard.nextLine();

The favorite values are stored in 6 local variables. The next step is store this information in a Favorites object so that it can be grouped together. To do this you instantiate a Favorites object and pass the favorite values to the object through the constructor's parameter list. Add the following code to the inputFavorites method.

myFavorites = new Favorites(name, food, color, music, auto, sport);
