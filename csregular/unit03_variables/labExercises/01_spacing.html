Spacing

In this lesson you are going learn about the concatenation operator "+" and how it can be used to enhance a programs output.

Recall that a string literal consists of characters placed inside a pair double quotes. Here are a few examples of string literals: "dog", "horse", "Sam ran fast.", "375", "5 + 7", "H", "Answer = ". The concatenation operator "+" is used in conjunct with strings to allow printing of data of multiple types.

Task - Add the following code to the Exercise 1 section, then execute the program.

System.out.println("Answer = " + 25);

In the parentheses of the println statement are two different types of data. "Answer = " is a string and 25 is an integer. Without the plus sign "+" (concatenation operator) printing these values in the same println statement would not be possible unless you place the number 25 inside the quotes of the string.

Task - Add the following code to the Exercise 1 section, then execute the program.

System.out.println("Answer = 25");

Although this code works, it is not always possible to combine the values in this manner.

Task - Add the following code to the Exercise 1 section, then execute the program.

System.out.println("Answer = " + num1);

Task - Add the following code to the Exercise 1 section, then execute the program.

System.out.println("Answer = num1");

You will notice this technique does not work when using variables instead of literal values.


Exercise 2

Task - Add the following code to the Exercise 2 section, then execute the program.

System.out.println(1 + 2 + 3 + 4 + 5);

This line of code calculates the sum of 1 + 2 + 3 + 4 + 5

Task - Add the following code to the Exercise 2 section, then execute the program.

System.out.println("Sum = " + 1 + 2 + 3 + 4 + 5);

Did it print what you expected? Since the println statement begins with the string "Sum = " the plus signs after the string no longer do addition, instead they perform concatenation which merges all the numbers into one string "12345".

Task - Add the following code to the Exercise 2 section, then execute the program.

System.out.println(1 + 2 + 3 + " " + 4 + 5);

This team the a string was placed between the 3 and 4, therefore the three plus signs leading up to the string are plus signs and do arithmetic. However, the two plus signs after the string are converted to concatenation operators which turns the two numbers into strings and merges them together with the sum calculated by the first three numbers.

Exercise 3

Task - Add the following code to the Exercise 3 section, then execute the program.

System.out.println(10 + " + " + 20 + " + " + 30 + " = " + (10+20+30));

This line of code displays an arithmetic problem along with the computer calculated result. The computer uses the order of operations or PEMDAS to determine the order the arithmetic expressions are evaluated. Since the last three numbers are wrapped inside parentheses the order of operations determines that they should be evaluated first so the plus signs are not converted to concatenation operators and the sum of the three numbers is calculated.

Task - Add the following code to the Exercise 3 section, then execute the program.

System.out.println(10 + " + " + 20 + " + " + 30 + " = " + 10+20+30);

This time the last three numbers do not include parentheses, therefore there plus signs become concatenation operators and the numbers are merged together.
