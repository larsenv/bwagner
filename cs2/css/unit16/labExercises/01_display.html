CSS Layout - Display Property

CSS Layout refers to the arrangement of text, images, and other elements on a web page. The display property is the most important CSS property for controlling layout.

The display property specifies if/how an element is displayed.

Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is block or inline.

  3. Add a class to the internal styling named block_container. Give the class the following styling:

    • Use the border property to style the border with the following attributes: border size of 1px, border style of solid, and border color of black.
    • Set background color to light gray.
    • Set height to 40px.
  4. Add a class to the internal styling named inline_container. Give the class the following styling:
    • Set line height property to 40px.
    • Set the text color to green.
  5. Add a class to the internal styling named testWidthHeight. Give the class the following styling:
    • Use the border property to style the border with the following attributes: border size of 1px, border style of solid, and border color of black.
    • Set width to 400px.
    • Set height to 200px.
    • Set background color to lightskyblue.
  6. Display the web page in a browser and observe how block and inline elements differ in the way they are displayed.
