Homeward Angel 2

Lesson

In this lesson you are going to rewrite the program HomewardAngel, done previously, so that the Jeroo's behavior is divided into the following 3 Jeroo methods:

  • getFlower - this method moves the Jeroo from its starting position to on top of the flower
  • pickAndToss - this method picks up the flower and tosses it at the net
  • enterHouse - this method moves the Jeroo through the opening in the net and finishes inside the house

Directions

  1. Copy the following starter code.

    method main()
{
   Jeroo angel = new Jeroo();
       


}

  2. Click on the save icon (look like a floppy disk). Navigate to your desktop and save the file with the name: HomewardAngel2.

  3. Click on the Island File Menu and select Open. Navigate to the Jeroo folder on your desktop. In the Jeroo folder open the Islands folder and select the island file Homeward Angel.

  1. User-defined methods in Jeroo are coded in the Jeroo methods section. Refer to the illustration below.

    Copy the following code and paste it into the user-defined method section.

    method getFlower()
{

}

method pickAndToss()
{

}

method enterHouse()
{

}

  2. Copy the following code and paste it in the main method below the code that creates the Jeroo angel.

    angel.getFlower(); 
angel.pickAndToss(); 
angel.enterHouse();

  3. Copy the following code that moves angel on top of the flower so he can pick it up. Paste the code inside the curly braces for the method getFlower().

    turn(RIGHT);
hop(3);
turn(LEFT);
hop(4);

    Notice the you do not put the Jeroo's name at the front of this code.

  4. Copy the following code that instructs angel to pick up the flower and toss it onto a net to disable it. Paste the code inside the curly braces for the method pickAndToss().

    pick();
turn(RIGHT);
hop();
turn(LEFT);
hop();
toss();

  5. Copy the following code that instructs angel to go home. Paste the code inside the curly braces for the method enterHouse().

    hop(4);
turn(LEFT);
hop(3);
