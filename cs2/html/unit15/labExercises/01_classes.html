15-1 HTML Classes

Resources

  • HTML Classes

Lesson

In this lesson you are going learn how to assign class names to HTML elements. Elements that are assigned or share the same class name will have the same styling.

The class attribute uses the same format as other html attributes.

<div 
class
="myClassName">


</div>

Lab Exercise

  1. Open VS Code and create a new document named 15_1_classes.html and save it to your HTML folder.

  2. Give the document the title: 16_1_classes.

  3. Copy the following HTML code into your new document.

  4. Display the html file in the browser. Notice there is a button named Submit at the top of the window. Click on this button. HTML provides a tag named <button> that can be used to add buttons to an HTML document. However, the styling of this button is somewhat plain.

    To demonstrate how classes work in HTML you are going to transform the five divs defined at the bottom of the page into buttons without using the HTML button element.

  5. The first thing you may notice is there is a lot of CSS styling going on between the two <style> tags. You will learn more about this code when you do the CSS section of the course, so for now, just ignore it. The code that relates to this lesson is highlighted below.

    <style> 
   
    .myButton
    
   {


   }
</style>

    In CSS, a class is defined by placing a period '.' in front of an identifier name. For example, this code is defining a class named myButton.

  6. Once a class has been defined it can be assigned to HTML elements by using the class attribute.

    Add
    the following HTML code as an attribute for the "Ok" div.

    <div
     class="myButton"
    >Ok</div>

    Refresh the browser window to view the changes. The "Ok" div should now be a button.

    As you can see, when you assign a class to an HTML element that element is displayed using the class's CSS styling.

  7. Assign
    the myButton class as an attribute to other four divs.

    Refresh the browser window to view the changes. All of the divs should now be buttons with the exact same styling.

Lesson Review

  • Classes provide an easy way for multiple HTML elements to have the same styling.

  • When declaring a class in CSS a period '.' is required in front of the class name.

  • In HTML classes are declared as attributes of HTML elements using the keyword "class".
