Lab Exercise

  1. Open VS Code and create a new document named 18_2_audio.html and save it to your HTML folder.

  2. Give the document the title: 18_2_audio.

  3. Use HTML5 audio elements to play the following audio within your document.

  4. Create a new folder inside your HTML folder named audios.

  5. Right-click the following link and save the mp3 file to your audios folder: Lou Gehrig.mp4

  6. Use the control and autoplay attributes.
