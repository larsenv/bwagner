CSS Background

In CSS, there are several background properties that allow you style a background color or image.

These background properties include

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

These background properties are most often applied to a page's body element but they can also be used with other elements such as divs.

In this lesson, you will explore some of these background properties. To get started click on the following link to learn more about these properties: CSS Background.

  3. By default, the property background-repeat is set to repeat. When an image is assigned to a background, if its size is smaller than the element's size then the image will be repeatedly rendered from top to bottom and left to right until it fills the entire background. You can turn off this feature by setting the property to no-repeat.

  4. Using internal styling, add the following css to your page's body element.

    background-image:url("image_name");

  5. Use the websites listed in the Resource section above to find or create your own background image. When you have found your image, save it to the same location as your html document.

  6. Change the name of image listed in the background-image property from "image_name" to the name of your background image.
